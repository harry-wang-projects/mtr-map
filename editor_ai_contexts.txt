editor_version1.html:

Focus on the folder @editor_version1.
This is a json editor that edits a json file (an object in the code) that stores information about a subway network. The network consists of lines, which consist of branches, which consist of stations. All information can be edited. In addition, the positions (latitude, longtitudes) of stations can be edited by clicking on a map.

Rules:

Task:

If the highlighted checkpoint is the last checkpoint of its station, then switch the highlighted object to the next station relative to the station the currently highlighted object is part of. If the currently highlighted object is a station, then if the station has checkpoints, change the highlighted object to the first checkpoint of the currently highlighted station. If the currently highlighted station doesn't have checkpoints, then highlight the next station. If the currently highlighted station is the last station, then go back to the first station.

If the highlighted object is a station, then switch the highlighed object to the checkpoint or station right before it. For all cases, do the opposite of when I press S. Instead of highlighting the next checkpoint or station, highlight the previous one.
- For when I press E, write a new function. Make it different from the current add_checkpoint function. Change it as follows: If a station that has no checkpoints is highlighed, then enter checkpoint mode like before, inserting a checkpoint to the station; If a station with checkpoints is highlighted, then enter checkpoint mode, inserting a checkpoint between the station and it's first checkpoint; If a checkpoint is highlighted, then enter checkpoint mode, inserting a checkpoint after the currently highlighted checkpoint but before the next one. If that checkpoint is the last checkpoint of its station, then simply insert another checkpoint after it.
